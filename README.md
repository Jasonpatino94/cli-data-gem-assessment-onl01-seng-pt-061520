Description:
Used to get facts about certain breed of dogs

Usage: 
1) To start program, type in "bin/run"
2) At this point you can press "1" to get a list of all dog breeds or "2" to exit program
3) You then get a list of all the dog breeds and you can either choose them by their index number OR typing some or all of their breed. 
4) If you type some of their breed name, you will get a list of all breeds similar to that name
